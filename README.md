# 🌍 Maharashtra Explorer

A modern, interactive web project showcasing the cities, culture, heritage, cuisine, and festivals of Maharashtra, India.

## ✨ Features

- 🏙️ Explore major cities of Maharashtra
- 🔍 Search & filter cities (Metro, Heritage, Coastal, Pilgrimage, Hill Stations)
- 🗺️ Interactive map using Leaflet.js
- 🏛️ Heritage section (UNESCO sites & forts)
- 🎉 Festivals & traditions showcase
- 🍛 Famous Maharashtrian cuisine section
- 📱 Fully responsive design
- ⚡ Smooth animations & modern UI
